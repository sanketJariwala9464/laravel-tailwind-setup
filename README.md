# Laravel + Tailwind Setup

A simple laravel + tailwind css setup.

## Installation

Use the package manager [composer](https://getcomposer.org/doc/00-intro.md) to install packages.

```bash
composer install
```

## Install Npm Packages

```bash
npm i or yarn install 
```

## Run tailwind CSS 

```bash
npm run tailwind 
```
